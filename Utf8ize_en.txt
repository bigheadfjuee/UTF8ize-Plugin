# UTF8ize Plugin

## What's this?

This plugin sets the source code file format to UTF-8 when creating a new file.

## Supported versions

2010 - XE10.3

## Installation

1. Open the package "Utf8ize.dpk"
2. Right click on "Utf8ize.bpl" and click "Install"
